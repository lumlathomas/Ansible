Ansible Include

Ansible include is used to modularize and reuse playbooks or tasks.

Types of Include

  • include_tasks: Includes tasks from another file.
  • include_playbook: Includes an entire playbook.

Example: include_tasks


- name: Main Playbook
  hosts: localhost
  tasks:
    - name: Include common tasks
      include_tasks: common_setup.yml

Example: include_playbook


- name: Main Playbook
  hosts: localhost
  tasks:
    - name: Include another playbook
      include_playbook: another_playbook.yml

Benefits

  • Reusability
  • Modularity
  • Maintainability
  • Readability

Run Playbook

ansible-playbook main_playbook.yml
